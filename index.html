🔑 Conectar la inteligencia artificial

Este prototipo clasifica los reportes usando un modelo de IA real (Claude). Para mantener tu clave segura, no la guardamos en ningún servidor: se usa solo desde tu navegador, en esta sesión. Pega tu API key de Anthropic para activar la clasificación.

¿Sin clave a mano? Puedes activar el modo demostración, que clasifica con reglas locales y permite probar todo el flujo sin conexión a la IA.
